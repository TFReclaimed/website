# Troubleshooting

Here you'll find solutions to common issues that may arise while playing Titanfall Frontline.

## Not getting a match despite plenty of players online

Some people just open the game and AFK in the menus. Use the `/who` chat command to see who's online and yell at them to come play.

You may want to send a message on the Discord server to see if anyone is up for a match.

## Desync in battles

The battle code is... not amazing. If you experience desync in battles, try restarting the game to reload the game state.

## Accepting casual match does nothing for one player

Sometimes when requesting a casual match both players aren't immediately sent into the match.

If you wait you will eventually load into the match, or you can skip the wait by restarting the game.

## Buttons are overlapping and game is zoomed in

The game is originally designed for a 9:16 aspect ratio and doesn't scale well to taller aspect ratios.

We've added letterboxing to help reduce this, however it's not perfect.

On wider aspect ratios (desktop) everything will render correctly.

## Text beginning with "[DNF:"

The original developers forgot to add a text for this translation key.
These are usually only seen in the missions.
