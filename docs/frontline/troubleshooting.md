# Troubleshooting

Here you'll find solutions to common issues that may arise while playing Titanfall Frontline.

## App isn't compatible with your device (INSTALL_FAILED_NO_MATCHING_ABIS)

Newer chipsets and Android versions have begun dropping support for 32-bit apps.

We're working on recompiling the game for 64-bit, but in the meantime you can virtualize a 32-bit Android environment using [VPhoneOS](https://play.google.com/store/apps/details?id=com.yoyo.snake.rush).

## Stuck in loading screen after finishing tutorial

You didn't download the patched version! Read the installation instructions again.

## Desync in battles

The battle code is... not amazing. If you experience desync in battles, try restarting the game to reload the game state.

## Game freezes after opening hangar

This mostly happens on emulators, but can also occur on real devices.

There is currently no fix for this issue, but if you keep trying it will eventually work.

## Accepting casual match does nothing for one player

Sometimes when requesting a casual match both players aren't immediately sent into the match.

If you wait you will eventually load into the match, or you can skip the wait by restarting the game.

## Store purchase button doesn't work

Make sure you haven't disabled the GMS patch in FrontlinePatcher.

## No text is being rendered

This is an issue on some devices. Use a different device.

## Buttons are overlapping and game is zoomed in

The game is designed for a 16:9 aspect ratio. Most devices don't have this aspect ratio anymore and try to resize the game to fill the screen.

How you disable this feature depends on your device, but look for options similar to `App full screen` or `Full screen apps`.

:::note

On some devices (Android 10 One UI) disabling seems to be completely broken and not do anything.

:::
