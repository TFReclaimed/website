# iOS

## Requirements

- iOS 12 or higher
- At least 500 MB of free storage space

## 1. Install SideStore

To install Titanfall Frontline on iOS, you will need to use SideStore, a third party app that allows you to sideload and resign apps directly on your device.
A computer is initially required to install SideStore.

If you already have some apps sideloaded on your device you may wish to additionally install LiveContainer to bypass the 3 app limit.
This is not required but highly recommended.

- [SideStore Installation Guide](https://docs.sidestore.io/docs/installation/prerequisites)
- [LiveContainer Installation Guide](https://livecontainer.github.io/docs/installation)

## 2. Download the IPA

Get the latest IPA by [clicking here](http://cdn-cf.tfflinternal.com/frontline/Frontline.ipa).

## 3. Install the IPA

### Using SideStore

1. Open SideStore on your iOS device.
2. Go to the "My Apps" tab and tap the "+" button in the top right corner.
3. Select the downloaded Frontline.ipa file from your device's storage.

### Using LiveContainer

1. Open LiveContainer on your iOS device.
2. Tap the "+" button in the top right corner.
3. Select the downloaded Frontline.ipa file from your device's storage.
