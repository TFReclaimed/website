# Main Menu

{/* Please update this area when Reputation becomes meaningful */}

Frontier Missions - Single player missions that reward XP, Tokens, Credits, and Exclusive Cards.

Battle - Online PvP matches with human players.

{/* Please update this area when Credits can purchase boosters */}

Black Market - Tokens are used to purchase Boosters Packs that contain 5 cards each.

Ranks - Leaderboard of top players based on Trophy count. Can issue challenges to online players.

Hangar - Contains acquired cards and resources and is used to build decks of cards called "Dropships".

Comms - Players can leave messages for Server and/or Guild. Game related notifications appear here.

Settings (Gear) - User ID, Adjust Music and SFX Volume, Help page, Credits page, and Logout Button.

Guild - Member roster. Can send hourly XP Gift, view stats, and issue challenges to online guild mates.
